# 锦绣超市进销存信息管理系统-后端

本项目是本科的毕业设计作品，主要是实现超市的进销存信息管理。采用前后端分离的开发方式，后端使用SpringBoot，前端使用React。

技术栈：SpringBoot + SpringCloudAlibaba + Mybatis + JWT + Redis + Mysql + logback + Maven + Swagger + Knife4j
